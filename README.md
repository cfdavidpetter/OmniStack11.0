### Install

#### Download of Project 📋
    git clone https://github.com/cfdavidpetter/OmniStack11.0.git
#### Backend 🛠
    cd backend && npm install && npm run start
#### Frontend 💎
    cd frontend && npm install && npm run start
#### Mobile 📱
    cd mobile && npm install && npm run start
